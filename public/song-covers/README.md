# Song Cover Images

Add your song cover images to this directory with the following filenames:

- `saturn.jpg` - SZA - Saturn
- `superposition.jpg` - Daniel Caesar - Superposition  
- `latch.jpg` - Sam Smith - Latch
- `somewhere-in-brooklyn.jpg` - Bruno Mars - Somewhere In Brooklyn
- `lavender-buds.jpg` - MF DOOM - Lavender Buds
- `love-me-not.jpg` - Ravyn Lenae - Love Me Not

## How to get song covers:

1. **Spotify/Apple Music**: Right-click on album art and save image
2. **Google Images**: Search for "[Song Name] [Artist] album cover"
3. **Music streaming services**: Most have high-quality album art
4. **Discogs/Last.fm**: Great sources for album artwork

## Image requirements:
- Format: JPG, PNG, or WebP
- Size: Square images work best (300x300px or larger)
- Quality: High resolution for crisp display

The images will be displayed as 96x96px squares with rounded corners and shadow effects.
